## Avalonia (for creating new projects)

Avalonia is used for the App UI.
Install it via: `dotnet new install Avalonia.Templates`.

## wasm-tools (for browser only)

wasm-tools is a set of tools to help with the development of Wasm.
Install it via: `dotnet workload install wasm-tools`.
