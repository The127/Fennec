## Codebase Patterns

- NodaTime `Instant.FromUtc` only accepts hours 0-23. For timezone-stable day-boundary tests, use midday UTC (12:00) on different calendar days rather than 23:59 → 00:00 crossing.
- Domain classes live in Fennec.App/Domain/. Tests live in Fennec.App.Tests/Domain/.
- ServerViewModel already imports `using Fennec.App.Domain;` — domain classes are ready to use.
- EmojiDatabase is in namespace Fennec.App.Models. Known shortcodes: "grinning" → U+1F600, "smiley" → U+1F603.
- ServerRoute is a record in Fennec.App/Routes/ServerRoute.cs — has ServerId and InstanceUrl properties directly.
- Formatting helpers (not domain logic) live in Fennec.App/Formatting/. Tests in Fennec.App.Tests/Formatting/.
- Message is a `readonly record struct` — use `Message.Create(...)` factory; throws `ArgumentException` on bad ISO timestamp.
- MessageGrouper methods require explicit `DateTimeZone zone` parameter; callers should pass `DateTimeZoneProviders.Tzdb.GetSystemDefault()`.
